# ld42: Running out of space
by [@fenderic](https://github.com/fenderic) & [@jhowcrof](https://github.com/jhowcrof)

----
## controls
* WASD to move
* Click to shoot bullets
* E to bomb
* Space Bar to use shield
* R to reset
